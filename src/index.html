Why do you need to
work with me?

I am a London based Junior Front-end Developer. I have a passion for creating fun, responsive, and dynamic websites, creating code that is readable, modular and organised. My approach is to start with a desktop-first design and media query my way to responsive mobile versions, with HTML, JavaScript, Sass, PHP, MySQL, I also work with Javascript libraries React, GSAP and dabble with animation libraries like Three.js.

Problem Solving

I approach each problem as a learning experience, identifying a complex problem, breaking it down into smaller components and solving it. I am equally driven when approaching problems as part of a team or working as an individual.

Creating pixel perfect layout allowing users to intuitively navigate through a website, with the use of media queries, CSS grid, and flexbox. I'm flexible with mobile-first and desktop, taking a design and building it out ready for commercial deployment.
